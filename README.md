# React Simple Dzikir Counter
My first React Js component, simple dzikir counter

[**DEMO**](http://react-dzikir-counter.herokuapp.com/)

# Usage
1. Clone this repo
2. Install dependencies
	```
	npm install
	```
3. Compile with
	```
	webpack
	```
4. Run server, the server listen in port 3000
	```
	node server.js
	```
